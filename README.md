# 📝 Notes App

A full-stack MERN Notes Application.

## Features

- User Registration
- User Login & Authentication
- JWT Authorization
- Create Notes
- Edit Notes
- Delete Notes
- User-specific Notes
- Responsive Dark UI

## Tech Stack

Frontend:
- React
- Vite
- Axios
- React Router

Backend:
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- bcryptjs

Frontend
